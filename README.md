# 김성은 Personal Portfolio Website

## Overview
A modern, responsive personal portfolio website showcasing my professional journey, skills, and projects.

## Technologies Used
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- React Scroll
- React Typed

## Features
- Responsive Design
- Smooth Scrolling Navigation
- Animated Typing Effects
- Modern UI/UX

## Getting Started

### Prerequisites
- Node.js (v14+)
- npm or yarn

### Installation
1. Clone the repository
2. Navigate to the client directory
3. Install dependencies
```bash
npm install
```

### Running the Project
```bash
npm start
```

## Deployment
```bash
npm run build
```
